# KinoBot
Привет! 👋🎬 Я КиноБот. Если ты хочешь найти фильм, посмотреть список отложенных или узнать о схожих фильмах, я готов тебе помочь! Просто воспользуйся кнопками ниже. Наши возможности включают поиск фильмов, отбор фильмов по критериям, сохранение понравившихся фильмов, а также просмотр информации о текущих и отложенных фильмах.
